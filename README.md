# Small colored logger.

### Usage

Import in and use as:

```
logger('Hello world')
or with color:
logger('ok', 'Hello world')
```

### Available colors:

`Node terminal colors`

-   `info` Cyan
-   `ok` Green
-   `warn` Red
-   `debug` Yellow/Orange
-   `silly` Magenta
-   `white` Default. Used by logger after coloring - `logger('info', 'This will be white')`
